

# GIT命令

### 查看用户名 

####  git config user.name

### 查看邮箱     

#### git config user.email

### 修改用户名

####  git config --global  user.name "用户名"

### 修改邮箱

#### git config --global user.email "邮箱" 

### 添加远程

####  git remote add origin “git路径”

### 管理远程

####  git remote remove origin
